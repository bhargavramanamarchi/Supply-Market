# 📌 Overview

Supply Market is an AI-powered B2B procurement platform where buyers describe their requirements using natural language or voice.

Google Gemini AI analyzes the requirement, searches the supplier database, recommends the most suitable suppliers, explains the recommendation, and allows buyers to connect directly with suppliers.

Supabase provides Authentication, PostgreSQL database, and Row Level Security.

Murf AI provides multilingual voice responses.

# 🗄️ Database & Security

### Database Tables

The application uses the following PostgreSQL tables in Supabase:

- **users**: Stores registered user profiles and role-based accounts (Buyer/Seller).
- **suppliers**: Stores verified supplier business profiles.
- **products**: Catalog containing details, pricing, quality grade, and stock level of materials.
- **buyerrequests**: Records natural language sourcing requirements submitted by buyers.
- **airecommendations**: Stores confidence match scores generated by Gemini matching logic.
- **buyer_connections**: Tracks connections requested between buyers and suppliers.
- **buyer_follows**: Logs supplier profiles bookmarked/followed by buyers.

### Security

- **Supabase Authentication**: Secure user login, signup, and persistent role-based sessions.
- **PostgreSQL**: Robust, transaction-safe backend relational storage.
- **Row Level Security (RLS)**: Enforces access control rules ensuring users only read or write their authorized data scope.
- **Protected APIs**: Serverless routes and external AI client connections require valid credentials.
- **Secure Environment Variables**: Keeps API keys hidden from client-side compilation exposures.

# ⚙️ Installation

Clone Repository:

```bash
git clone https://github.com/YOUR_USERNAME/Supply-Market.git
```

Move into Project:

```bash
cd Supply-Market
```

Install Dependencies:

```bash
npm install
```

Create `.env`:

```env
VITE_GEMINI_API_KEY=your_gemini_api_key
VITE_MURF_API_KEY=your_murf_api_key

VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

Run Project:

```bash
npm run dev
```
